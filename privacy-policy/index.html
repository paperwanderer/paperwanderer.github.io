🔒 개인정보처리방침

최종 업데이트: 2026년 4월 4일

1. 소개

paperWanderer(이하 “블로그” 또는 “서비스”)는 이용자의 개인정보보호를 매우 중요하게 생각합니다. 이 개인정보처리방침은 본 블로그가 수집하는 정보, 그 정보의 사용 방법, 그리고 이용자의 권리에 대해 설명합니다.

2. 수집하는 정보

2.1 자동으로 수집되는 정보

  • 구글 애널리틱스(Google Analytics): 블로그 방문 통계, 페이지 뷰, 방문 경로, 기기 정보, 대략적인 위치 정보 등
  • 쿠키 및 유사 기술: 블로그 기능 개선 및 이용자 경험 향상을 위한 쿠키
  • Google AdSense: 광고 서빙 및 성능 추적을 위한 정보
  • 서버 로그: IP 주소, 브라우저 종류, 접속 시간, 요청 페이지 등

2.2 이용자가 제공하는 정보

  • GitHub Issues를 통한 의견, 제보, 문의사항 (선택적)
  • 이메일을 통한 직접 연락 (선택적)

3. 정보 수집의 목적

  • 블로그 운영 및 서비스 제공
  • 사용자 경험 개선 및 콘텐츠 최적화
  • 접속 통계 분석 및 트래픽 모니터링
  • 광고 서빙 및 광고 수익 창출
  • 보안 및 부정 사용 방지
  • 법적 요구사항 충족

4. 정보 공유 및 제3자 제공

본 블로그는 다음의 제3자 서비스 제공자와 정보를 공유합니다:

  • Google LLC: Google Analytics, Google AdSense, Google Fonts 등 - Google 개인정보처리방침 참고
  • GitHub: 블로그 호스팅 및 이슈 관리 - GitHub 개인정보처리방침 참고

이용자의 동의 없이 다른 제3자에게 개인정보를 판매하거나 공개하지 않습니다.

5. 쿠키 정책

본 블로그는 쿠키를 사용합니다:

  • 필수 쿠키: 블로그 기능 정상 작동을 위해 필요
  • 분석 쿠키: Google Analytics를 통한 사용 통계 수집
  • 광고 쿠키: Google AdSense를 통한 맞춤형 광고 제공

이용자는 브라우저 설정을 통해 쿠키를 거부할 수 있습니다. 다만, 일부 기능이 정상 작동하지 않을 수 있습니다.

6. 정보 보안

본 블로그는 이용자의 개인정보를 보호하기 위해 적절한 기술적, 관리적 보안 조치를 취하고 있습니다. 그러나 인터넷 전송 과정에서의 보안을 100% 보장할 수 없습니다.

7. 정보 보유 및 삭제

  • Google Analytics: 기본적으로 25개월간 보유 후 자동 삭제
  • 쿠키: 브라우저 설정에 따라 자동 삭제
  • GitHub Issues를 통한 문의: 이용자 요청 시 삭제 가능

8. 이용자의 권리

이용자는 다음의 권리를 가집니다:

  • 접근권: 본인의 개인정보 열람 요청
  • 정정권: 부정확한 정보의 수정 요청
  • 삭제권: 개인정보 삭제 요청
  • 처리 제한권: 정보 처리의 제한 요청
  • 거부권: 마케팅 관련 정보 수신 거부

이러한 권리를 행사하려면 아래의 연락처로 문의해주세요.

9. Google AdSense

본 블로그는 Google AdSense를 통해 광고를 게재합니다. Google AdSense는 이용자의 관심사에 기반한 맞춤형 광고를 제공하기 위해 쿠키와 같은 추적 기술을 사용합니다.

  • Google 광고 정책
  • Google 광고 설정 — 광고 개인화 설정 변경

10. 약관 변경

본 개인정보처리방침은 언제든 변경될 수 있습니다. 중요한 변경사항이 있을 경우 본 페이지에 공지하겠습니다.

11. 연락처

개인정보 관련 문의사항이 있으시면 다음을 통해 연락해주세요:

  • 문의: GitHub Issues
  • GitHub Issues: 의견 · 제보 · 말 걸기

문의를 주신 후 비공개로 연락을 원하시는 경우, GitHub Issue에 이메일 주소를 남겨주시면 제가 먼저 연락드립니다.
